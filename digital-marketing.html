Digital Literacy Course

Learn essential digital skills to navigate computers, internet, and online tools confidently. Perfect for beginners aiming to enhance their technology knowledge.

Level 1: Beginner

Module 1: Understanding Computers

  • Types of devices: Desktop, Laptop, Tablet
  • Basic hardware components and their functions
  • Introduction to Operating Systems (Windows, Linux)

Module 2: Basic Software Skills

  • Using Word Processors: Create, format, and save documents
  • Spreadsheets basics: Entering data and simple calculations
  • Introduction to presentations

Module 3: Internet & Communication

  • Web browsing, searching efficiently
  • Creating and managing email accounts
  • Introduction to cloud storage tools (Google Drive, OneDrive)

Level 2: Intermediate

Module 1: File & Data Management

  • Organizing files and folders efficiently
  • Using external drives and cloud storage for backup
  • Basic data entry and handling techniques

Module 2: Productivity Tools

  • Advanced Word features: templates, headers, footers
  • Spreadsheets: formulas, charts, and simple analysis
  • Presentation tips: slides, multimedia, transitions

Module 3: Digital Collaboration

  • Google Docs, Sheets, and Slides for teamwork
  • Sharing files and setting permissions
  • Basic online productivity apps and tools
